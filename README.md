# FYP-Support-Ticketing-System

MONGODB setup
1. `DOWNLOAD MONGO FROM `https://www.mongodb.com/download-center/community `CHOOSE SERVER`
2. `PICK YOUR SYSTEM AND LATEST VERSION`
3. WINDOWS ONLY SCRIPT: `"path\to\MongoDBServer\bin\mongod.exe" --dbpath "path\to\MongoDBServer\data\db"
# FILES

0. Make sure MONGODB is running

1. `npm install`
2. `cd frontend-client`
3. `npm install`
4. `cd ../`
5. `npm run both`


selenium setup

1. `Create folder on C drive, call it Selenium`
2. `Put the downloaded driver into that folder`
3. `Go to system variables`
4. `Go to environment variables`
5. `Click on PATH then EDIT then NEW then add C:\Selenium`
6. `Now to see if it works, go to CMD and type start chromedriver`
7. `It should start the chrome driver`
